Você deve estar se perguntando o motivo para usar uma função lambda. A função lambda é feita para reduzir linhas de código. Contudo, entenda uma coisa: só funciona para funções pequenas

Nesta função que criei, a ideia era fazer o cubo de um número. Simples não é? Apenas colocar ** 3 e a sua função estava feita.

Contudo, podemos reduzir todo este código para apenas 3 linhas.

Primeiro você dá o nome da sua função como se fosse uma variável. É isso mesmo, só que ao invés de armazenar algo, em seguida você vai escrever lambda para identificar que é uma função lambda.
Em seguida, você dá o parâmetro como se fosse fazer na def simples e como é a expressão (como a função deve se comportar) e tcharam. Sua função lambda está pronta!
